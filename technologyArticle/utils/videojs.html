# videojs

# 契子：项目需求

公司项目之前一直有文件系统，现在有有个需求就是要在线播放视频

在网上一阵搜索，找到了几个方案、对比下来，感觉H5最新的标签<video>就很合适，直接引入标签用了，感觉样式有一些少，最后发现videojs是在video的基础上扩展的。

# 使用：Vue环境

因为项目主要是用Vue写的，所以在此处主要介绍vue里面的写法，如果还需要其他的，可以参考官网->Guides

videojs官网文档

  • # 1.安装

npm install video.js

  • # 2.vue cli中 使用(videojs官方文档)

<template>
    <div>
        <video ref="videoPlayer" class="video-js"></video>
    </div>
</template>

<script>
import videojs from 'video.js';

export default {
    name: "VideoPlayer",
    props: {
        options: {
            type: Object,
            default() {
                return {};
            }
        }
    },
    data() {
        return {
            player: null
        }
    },
    mounted() {
        this.player = videojs(this.$refs.videoPlayer, this.options, function onPlayerReady() {
            console.log('onPlayerReady', this);
        })
    },
    beforeDestroy() {
        if (this.player) {
            this.player.dispose()
        }
    }
}
</script>

因为是点击文件然后弹出视频，所以用了element-ui中的el-dialog组件

WARNING

在用el-dialog组合<video>时候的我遇到坑
给el-dialog标签中加上v-if，不然关闭el-dialog窗口后，视频还是会播放
初始化<video>标签时候，需要用this.$nextTick包裹初始化函数，不然会报错初始化没有 elementId
video标签中的source一定要把type加上，对应的文件加对应的type mp4 ->video/mp4，不然会报错

以下为模拟项目代码

// vue-video-player.vue
<template>
    <el-dialog
        v-if="visible"
        title="提示"
        :visible.sync="visible"
        width="50%"
        :close="handleClose"
        >
        <video ref="videoPlayer" class="video-js">
            <source :src="videoSrc" :type="`video/${extName}`">
        </video>
    </el-dialog>
</template>

<script>
  export default {
    props: {
        videoFlag: {
            type: Boolean,
            default: false
        },
        videoSrc: {
            type: String,
            default: false
        },
        extName: { // 文件的格式
            type: String,
            default: ''
        }
    }
    watch: {
        videoFlag (val) {
            this.visible = val
        },
        // 监听src 如果有src 才初始化video
        videoSrc (val) {
            if (val) {
                this.initVideo()
            }
        },
    },
    data() {
      return {
        player: null,
        visible: false,
        videoOptions: {
				autoplay: true,
				controls: true
			}
      };
    },
    methods: {
      handleClose() {
        this.$emit('close', false)
        // 关闭兑换框时候销毁video组件
        if (this.player) {
            this.player.dispose()
        }
      },
      initVideo () {
          this.$nextTick(() => {
              this.player = this.$video(this.$refs.videoPlayer, this.options, function onPlayerReady() {
                console.log('onPlayerReady', this);
              })
          })
      }
    }
  };
</script>
